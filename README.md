# Certify

Simple SSL certificate information library

## Install

Via Composer

``` bash
$ composer require bertuss/Certify
```

## Usage

``` php
$certify = new /bertuss/Certify/Certify('example.com');

$certify->isValid();
```

## Testing

``` bash
$ composer test
```
